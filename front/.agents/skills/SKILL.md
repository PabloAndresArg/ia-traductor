# Skill Instructions

## Habilidades Clave
- Comprensión de JSX y renderizado de componentes
- Manejo de estado con useState y useReducer
- Uso de useEffect para efectos secundarios
- Composición y reutilización de componentes
- Manejo de props y prop drilling
- Uso de Context API para manejo de estado global
- Implementación de hooks 
personalizados
- Manejo de formularios y validación
- Testing de componentes con herramientas como React Testing Library y Jest
- Optimización de rendimiento (memoización, React.memo, useCallback, useMemo)
- Accesibilidad (a11y) en componentes
- Uso de herramientas de desarrollo (React DevTools, ESLint, Prettier)

## Buenas Prácticas
- Mantener los componentes pequeños y enfocados en una sola responsabilidad
- Nombrar los componentes y archivos de manera descriptiva
- Evitar la mutación directa del estado
- Usar keys únicas y estables en listas
- Separar lógica y presentación cuando sea posible
- Documentar los componentes y hooks personalizados
- Escribir pruebas para componentes y lógica crítica
- Utilizar prop-types o TypeScript para validación de props
- Mantener la estructura del proyecto organizada
- Actualizar dependencias regularmente
- Separar cada componente en un archivo diferente
- Evitar que las funciones tengan más de 60 líneas
- Evita estilos globales de css a menos que sea necesario
